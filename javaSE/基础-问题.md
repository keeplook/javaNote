### 自增变量

下面结果多少：

```java
public static void main(String[] args) {
        int i = 1;
        i = i++;
        int j = i++;
        int k = i++ + i * i++;
        System.out.println("i=" + i);
        System.out.println("j=" + j);
        System.out.println("k=" + k);
    }
```
